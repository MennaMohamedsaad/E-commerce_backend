# E-commerce_backend
A backend system for an e-commerce platform built using Node.js, Express, and MongoDB. It includes user authentication, product and order management, and secure APIs for handling online store operations.
Key Features
  User registration, login, and authentication (JWT-based)
  Role-based access control (Admin & Customer)
  Product management (CRUD operations)
  Order creation and tracking
  Secure password hashing using bcrypt
  MongoDB with Mongoose for data storage
  Error handling and validation middleware

Technologies Used
  Node.js
  Express.js
  MongoDB & Mongoose
  JSON Web Token (JWT)
  Bcrypt
  Postman for API testing
